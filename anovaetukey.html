ANOVA e teste de Tukey

Página única em HTML + JavaScript para DIC e DBC, com entrada manual dos dados. Cada linha representa um tratamento e cada coluna representa uma repetição no DIC ou um bloco no DBC.

Entrada dos dados

Você pode editar os nomes dos tratamentos. Para DBC, cada coluna é tratada como um bloco.

Observações

Esta implementação foi pensada para dados balanceados. O teste de Tukey usa uma aproximação prática para o valor crítico da distribuição studentizada. Para uso acadêmico formal ou publicação, vale validar os resultados com um software estatístico de referência.
